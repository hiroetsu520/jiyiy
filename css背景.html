背景颜色

默认是transparent : 　背景色透明

背景图片

background-image。实际开发常见于logo或者一些装饰性的小图片或者是超大背景图片。
优点是非常便于控制位置（精灵图也是一种运用场景）
注意不要少了 url()

背景平铺

repeat : 　背景图像在纵向和横向上平铺
no-repeat : 　背景图像不平铺
repeat-x : 　背景图像在横向上平铺
repeat-y : 　背景图像在纵向平铺

背景位置（方位名词）

注意：也可以带精确单位
如果指定的两个值是精确单位和方位名词混合使用，则第一个值是x坐标，第二个值是y坐标（其他基本也是一样）

背景滚动/固定

注意：背景图默认是滚动的（scroll）
background-attachment
scroll : 　背景图像是随对象内容滚动
fixed : 　背景图像固定
语法：
html {
background-image: url("anasazi.png");
background-attachment: fixed;
}

背景半透明

背景半透明
语法：background: rgba(0,0,0,0.1);
1.最后一个参数时alpha透明度，取值范围时0-1之间
2.我们习惯性的把0.1的0省略掉，写为background: rgba(0,0,0,.1);
3.注意：背景半透明是指盒子背景半透明，盒子里的内容不受影响
4.css3新增属性，是ie9+版本浏览器才支持的
5.实际开发中我们不太关注兼容性写法了，可以放心使用

背景符合写法

为了简化背景属性的代码，我们可以将这些属性合并简写在同一个属性background中。从而节约代码量。
当使用简写属性时，没有特定的书写顺序，一般习惯为：
background:背景颜色 背景图片地址 背景平铺 背景图像滚动 背景图片位置;
